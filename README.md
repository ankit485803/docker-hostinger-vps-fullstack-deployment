# Docker Hostinger VPS Fullstack web application Deployment

End-to-end deployment of a full-stack web application using Docker containers on a Hostinger VPS, covering containerization, orchestration, and production setup. Inspired by learning from Chai Aur Code.

---

## Tech Stack
- Docker
- Docker Compose
- VPS (Hostinger)
- Full Stack (Frontend + Backend + Database)

---

## Features
- Containerized full-stack application
- Multi-service orchestration using Docker Compose
- Deployment on VPS server
- Production-ready setup
